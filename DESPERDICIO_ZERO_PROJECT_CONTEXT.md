# Desperdicio Zero - Project Context

## Visao
Plataforma digital para reducao de desperdicio de alimentos em cozinhas comerciais e domesticas.

## MVP Atual
- Area publica com demonstracao
- Cadastro e login (cliente/admin)
- Area do cliente
- Area administrativa
- Gestao de estoque
- Sugestoes de receitas
- Lista de compras
- Dicas e desafios

## Arquitetura Alvo
- Frontend React com Vite
- TailwindCSS
- Persistencia local no MVP (LocalStorage)
- Futuro: Firebase ou Supabase

## Roadmap
1. MVP web
2. Banco online
3. Multiusuario real
4. App mobile
5. Delivery inteligente
